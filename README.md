Fake News Headline Generator
<br>
Description
<br>
The Fake News Headline Generator is a simple Python project that randomly generates funny and unrealistic news headlines by combining predefined subjects, actions, and objects.
<br>
This project is beginner-friendly and helps in understanding:
<br>
Python lists and strings
<br>
Random module
<br>
Basic program logic
<br>
It is created mainly for learning and practice purposes.
<br>
Technologies Used
<br>
python random module
<br>
How It Works
<br>
The program stores different words in lists
<br>
It randomly selects words from each list
<br>
A fake news headline is generated every time the program runs
<br>
Purpose
<br>
Practice Python fundamentals
<br>
Understand random data generation
<br>
Build confidence in small Python projects
