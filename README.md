<h1>Fake News Headline Generator</h1>

<h2>Description</h2>
<p>
This project is a Fake News Headline Generator built using Python.
It randomly generates funny and unrealistic news headlines by combining
predefined subjects, actions, and objects.
</p>

<p>
This project is beginner-friendly and is useful for understanding
random data generation and basic Python programming concepts.
</p>

<h2>Technologies Used</h2>
<ul>
  <li>Python</li>
  <li>random module</li>
</ul>

<h2>How It Works</h2>
<ol>
  <li>The program stores words in multiple lists</li>
  <li>A random word is selected from each list</li>
  <li>A fake news headline is generated</li>
</ol>

<h2>Purpose</h2>
<p>
The main purpose of this project is to practice Python basics such as
lists, strings, functions, and the random module.
</p>
